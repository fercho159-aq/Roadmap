1. Comentarios en Python

Python ofrece dos formas de crear comentarios:

# Comentario de una línea

"""
Comentario de
múltiples líneas
usando triple comilla
"""

'''
También se pueden usar
comillas simples triples
para comentarios multilinea
'''

2. Variables y Constantes

En Python, las variables se crean simplemente asignando un valor:

# Variable
nombre = "Python"

# En Python no existen constantes verdaderas, pero
# por convención se usan mayúsculas para indicar que
# una variable debe tratarse como constante
PI = 3.14159
MAX_USUARIOS = 100

3. Tipos de Datos Primitivos

Python tiene varios tipos de datos primitivos:

# Strings (cadenas de texto)
texto = "Hola Python"
otro_texto = 'También con comillas simples'

# Números enteros (int)
edad = 25
numero_negativo = -10

# Números flotantes (float)
precio = 19.99
temperatura = -5.6

# Booleanos (bool)
es_verdadero = True
es_falso = False

# None (similar a null en otros lenguajes)
valor_nulo = None

# Complejos
numero_complejo = 3 + 4j

4. Hola Mundo

El clásico ejemplo "Hola Mundo" en Python:

print("¡Hola, Python!")

Sintaxis Básica

Características importantes de la sintaxis de Python:

  • Python usa indentación para definir bloques de código
  • No se necesitan punto y coma al final de las líneas
  • Los bloques se definen con dos puntos (:)
# Ejemplo de sintaxis con un if
if True:
    print("La indentación es importante")
    if True:
        print("Otro nivel de indentación")
print("Vuelta al nivel inicial")
